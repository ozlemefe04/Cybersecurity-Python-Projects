# Basic Cryptography Algorithms

This repository contains my first implementations of classic cryptographic algorithms using **Python**. As an **Aerospace Engineering** student with a passion for **Cybersecurity**, I am exploring the mathematical foundations of secure communication.

## Features
- **Caesar Cipher:** A simple substitution cipher that shifts characters by a fixed value. Includes input validation and handles both uppercase and lowercase letters.
- **Vigenère Cipher:** A polyalphabetic substitution cipher that uses a keyword for improved security. 

## Skills Demonstrated
- Python Programming (Control Flow, Loops, Error Handling)
- Modular Arithmetic (The core of cryptography)
- String Manipulation

## Context
These projects were developed as part of my learning journey on the **Cisco Networking Academy** (Python Essentials 2).
